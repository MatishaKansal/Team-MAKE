# Department-Based News Classification & Personalization System

## Overview

This project is a full-stack web application that uses Machine Learning to automatically classify news articles into predefined departments (e.g., CSE, Mechanical, Civil, etc.) and deliver personalized news feeds to users based on their selected department and interests.

The system integrates:
- React (Frontend)
- Node.js + Express (Backend)
- Python + FastAPI (ML Service)
- MongoDB (Database)

News articles are fetched periodically, classified using NLP techniques, and filtered so users only see relevant content.

---

## Key Features

- User Authentication
- Department-Based Filtering
- Automated News Updates (Every few Hours)
- ML-Based Text Classification
- Personalized News Feed
